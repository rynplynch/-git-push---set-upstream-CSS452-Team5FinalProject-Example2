API: Key Frame

Our module facilitates the animation creation process. The API is responsible to hold all animations associate with the object, as well as play/pause the animation. This allow one renderable object to have multiple animations and it can play any of its animations by setting its active animtion.
User only need to call newAnimation and then pass in object current state and their time in the animation to create the animation.
